### Added

- Symphony Orchestrator for GitHub Issue polling.
- Lobster Pipeline Engine for deterministic workflows.
- Core workflows: `issue-review`, `get-user-approve`, `decompose-issue`, `update-issue`.
- Mock-based test suite with zero external dependencies.
